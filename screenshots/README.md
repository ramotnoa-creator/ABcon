# Screenshots

Store UI screenshots here for reference and documentation.

## Naming Convention

```
YYYY-MM-DD-feature-description.png
```

Examples:
- `2026-02-11-tender-form-desktop.png`
- `2026-02-11-budget-variance-rtl.png`
- `2026-02-11-login-screen-hebrew.png`

## Note

Legacy screenshots may exist in `/Screenshoots/` (misspelled). New screenshots should go here.
