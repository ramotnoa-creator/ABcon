# ANcon (ABcon) - Construction Project Management

## What Is This?

A web-based construction and real estate project management system built for Israeli construction companies. Provides Hebrew RTL interface for managing projects, financial workflows, professional networks, and documentation.

## Who Uses It?

- **Project Managers (PM):** Manage projects, create estimates, handle tenders, track budgets
- **Entrepreneurs/Developers:** View project status, approve payments, review reports
- **Accountants:** Manage payments, track budgets, generate financial reports
- **Site Supervisors:** View tasks, update progress, access project documents

## Core Features

1. Project management (create, track, manage construction projects)
2. Financial management (estimates, tenders, budgets, payments, cost control)
3. Professional network (contractors, suppliers, service providers)
4. Task and milestone tracking (Gantt import)
5. Document management and export (PDF, Excel, Word)
6. Role-based access control (4 roles with granular permissions)

## Project Status
